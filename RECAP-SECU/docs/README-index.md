# Top10 OWASP

Nous allons passer en revue les **10 problèmes de sécurité les plus couramment rencontrés dans les applications web**, en suivant le [Top10 OWASP dans sa version 2021](https://owasp.org/Top10/fr/).

Pour chaque problème de sécurité potentiel, nous allons :

- **analyser / comprendre** ce problème
- **tester** si notre application est concernée
- **remédier** au problème, faire en sorte que notre application ne soit plus concernée !
- (et **tester à nouveau** pour s’en assurer !)
